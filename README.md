# Test EVV – Merge Game Demo

Небольшой прототип мобильной игры с **merge**‑механикой для Android.  
Проект создан на Unity и демонстрирует базовую логику объединения объектов в сетке (merge‑3 / merge‑2 стиль).

## Особенности

- Простая и понятная merge‑механика.
- Игровое поле в виде сетки с ячейками.
- Постепенное усложнение за счет появления новых типов объектов.
- Сборка под Android в виде готового APK.

## Установка на Android

1. Скачайте APK-файл из репозитория:
   - [`Test_MiniIT_1.0.apk`](builds/Test_MiniIT_1.0.apk)
2. Перенесите файл на Android‑устройство (или скачайте его прямо на телефон).
3. Разрешите установку приложений из неизвестных источников в настройках устройства.
4. Откройте APK-файл и подтвердите установку.
5. Запустите игру из списка приложений.
6. Также можно установить через adb install с ПК

## Как играть

- Перетаскивайте одинаковые элементы и объединяйте их, чтобы получить более высокий уровень.
- Новые объекты появляются автоматически на пустых клетках, когда они есть.

## Структура проекта

- `Assets/` – игровые ресурсы, сцены, скрипты.
- `builds/` – готовые сборки (APK и другие возможные билды).
- `ProjectSettings/` – настройки Unity-проекта.

## Требования

- Unity (версия указана в настройках проекта или в файле `ProjectVersion.txt`).
- Android‑устройство с поддержкой установки APK из внешних источников.
